## 💼 What I've Built

### 🤖 Life Sciences Multi-Agent AI Chatbot
Built and deployed an end-to-end multi-agent AI chatbot for the life sciences domain. Routes user questions to specialized sub-agents using **LangGraph**, with **RAG** pulling relevant domain knowledge into each response.
> FastAPI · LangGraph · LangChain · Weaviate · RAG · Streamlit · Python

---

### 🏗 Medallion Architecture Data Lake (Optum)
Designed a Bronze/Silver/Gold data lake on **Amazon S3** using **Delta Lake**, processing 10M+ daily healthcare records with full data quality checks, schema validation, and deduplication at each layer.
> AWS Glue · EMR · PySpark · Delta Lake · Airflow · Snowflake · GitHub Actions

---

### 🔄 CDC-Based Incremental Ingestion Pipeline (AT&T via HGS)
Built a real-time incremental ingestion platform using **CDC with AWS DMS**, keeping Redshift in sync with source system changes. Added automated data quality monitoring with alerts via Amazon SNS.
> AWS DMS · CDC · Redshift · AWS Glue · SNS · KMS · IAM · Python

---

### ✈️ Baggage Arrival Prediction — American Airlines
Processed **51M+ records** to build ML models predicting baggage arrival patterns at DFW Hub using XGBoost and HistGradientBoosting with multi-output regression.
> Python · XGBoost · Scikit-learn · Pandas · Time-Series · HPC

---

### 🏥 Claims Data Migration to Redshift (Cigna Health via HGS)
Migrated claims and member data from SQL Server, MySQL, and flat files into Amazon Redshift — with row count and checksum validation at every stage. Improved query performance by **50%** through Redshift tuning.
> Redshift · AWS DMS · Iceberg · Athena · Airflow · Python
